PackImpl Class Reference

Detailed Description

Base class template, this class is calculating the indices list by recursing over the TypeTuple while counting upwards (till I equals len(Tuple)/2 When it has finished recursing, the tail::index_values is equal to indices<0,2,4,...,len(Tuple) - 2>. By defining the index_values as Next::index_values, the top index_values will become equal to the tail index_values.

The return type will be equal to a tuple of pairs, the pairs will have the same first_type and second_type as the 0, 2, 4th ...element of the TypeTuple
